std1
	- TCP 간단한 통신 예제 1 (클라이언트 --> 서버 편도)

std2
	- TCP 간단한 통신 예제 2 (클라이언트 <-> 서버 양방향)

std3
	- UDP 간단한 통신 예제 (클라이언트 --> 서버)

std4
	- std2/std3 통합 버전

std5
	- std1에서 서버 : thread 추가 버전
	- std1에서 클라이언트 : 파일에서 읽어서 계속 보내도록 에뮬레이터

std6
	- std5에서 서버 : Non-Blocking mode

std7
	- std3에서 서버 : 비동기 입/출력 (signal : Linux only) ==> cygwin 터미널에서는 실행안됨.
